# 体重档案

一款可以记录和查询体重的应用，功能介绍如下

> * 支持体重数据的添加、删除、查询等功能
> * 动态生成体重曲线图，支持手势缩放
> * 实时计算BMI值，并进行诊断，给出健康体重范围
